# @gander-tools/diff-voyager-shared

Shared TypeScript types and utilities for Diff Voyager - a local website version comparison tool.

## Overview

This package contains shared domain models and TypeScript types used across the Diff Voyager monorepo:
- Backend (crawler and API)
- Frontend (Vue.js UI)

## Installation

```bash
npm install @gander-tools/diff-voyager-shared
```

## Usage

```typescript
import type { Project, Run, PageSnapshot } from '@gander-tools/diff-voyager-shared';
```
